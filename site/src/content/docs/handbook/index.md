Welcome to the **mcp-examples** handbook. This repo contains runnable example workspaces that teach the MCP Tool Shop model — read-only by default, side effects always opt-in.

## What is mcp-examples?

mcp-examples provides hands-on workspaces you can clone and run immediately. Each workspace teaches one concept clearly, with no boilerplate to wade through.
